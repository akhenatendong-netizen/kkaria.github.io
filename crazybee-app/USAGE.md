# Usage Guide

## 1. Open the full prototype
- Open `index.html` in your browser.
- It renders all screens in iPhone 15 Pro frames.

## 2. Open individual screens
- Open any page directly, for example:
  - `home.html`
  - `search.html`
  - `deal-detail.html`

## 3. Interaction behavior
- Tap cards and buttons to navigate.
- Tab bar switches between primary sections.
- Favorite icon toggles saved state with toast feedback.
- Segmented controls and chips switch active states.

## 4. Files
- `styles.css`: global visual system and layout components.
- `app.js`: shared navigation and UI interaction logic.
- `prototype-spec.md`: UX, IA, and implementation mapping.
