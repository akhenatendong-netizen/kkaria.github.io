# CrazyBee - North America Local Deals Prototype

CrazyBee is a high-fidelity mobile prototype for a Groupon/Dianping-style local deals app designed for the North American market.

## Highlights
- Full iPhone 15 Pro style viewport (393x852)
- iOS-like status bar and persistent bottom tab bar
- End-to-end user flow across 7 core screens
- Real Unsplash images (no placeholders)
- Shared visual system in `styles.css`
- Shared interactions in `app.js`

## Core Screens
- `home.html`
- `search.html`
- `restaurant-detail.html`
- `deal-detail.html`
- `orders.html`
- `favorites.html`
- `profile.html`
- `index.html` (all-screen preview wall)

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Font Awesome (CDN)
- Vanilla JavaScript
- Custom CSS design system

## Design Principles
- White surfaces + light gray background
- Brand red `#FF3535` for key actions only
- Rounded cards, subtle borders, soft shadows
- Spacious layout and clear visual hierarchy

## Run
Open `index.html` in a browser to preview all screens.
