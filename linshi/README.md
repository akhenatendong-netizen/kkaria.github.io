# 寄旅 AI 存包原型

这是一个可直接打开的高保真移动端原型，聚焦以下场景：

- 动态行程下的 AI 存包推荐，而不是单纯按当前位置推荐
- 整合智能柜、商家寄存、地铁站、火车站等存包能力
- 同时比较价格、绕路时间、营业时间和回取风险
- 支持柜体状态可视化、重量监测、异常取件提醒
- 支持物品类型识别后推荐专用寄存空间

## 运行方式

直接用浏览器打开以下任一页面即可，每个页面都有独立 URL：

- `index.html`
- `plan.html`
- `detail.html`
- `assistant.html`
- `orders.html`
- `profile.html`

## 说明

当前工作区没有实际业务接口，也无法直接读取受权限限制的 Google 文档和缺失的 `执行概要.pdf`，因此本版本基于你提供的产品描述完成了完整交互原型与视觉方案，可继续在此基础上接 API、拆 React/Vue 组件或接入地图与支付能力。

## 多页面说明

原型已拆成多页面结构，侧栏、底部 Tab 和关键按钮都使用真实链接跳转，便于后续接入正式路由或逐页开发。
